# GFNX: Generative Flow Networks in jaX
GFlowNet Environments in Jax

Installation is handled by `hatch`: https://hatch.pypa.io/latest/

Installtion: 
```
    pip install -e .
```
To install packages for algorithms, use
```
    pip install -e '.[algs]'
```
